# To Build and Run:

Run the following:
## Terminal 1:

```
$ python -m SimpleHTTPServer
```
## Terminal 2:

```
$ webpack --progress --colors --watch
```

## Browser
Go to `http://localhost:8000/`

For more information about running files locally, visit https://github.com/mrdoob/three.js/wiki/How-to-run-things-locally
